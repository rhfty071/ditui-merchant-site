# 地推服务商官网 - 网站定位

## 官网定位
**面向商户的地推服务接单官网**（B端）
- 目标用户：有地推需求的各行业商户（餐饮/教育/美容/零售/银行/商场等）
- 核心诉求：商户主动咨询 → 了解服务 → 提交需求 → 达成合作

## 页面结构

### 核心页面
- `/` - 首页：服务简介 + 核心卖点 + 咨询入口 + 成功案例
- `/services` - 服务介绍：服务类型 + 流程 + 报价
- `/cases` - 成功案例：案例列表 + 行业分布
- `/solutions/[industry]` - 行业解决方案（17个行业）
- `/contact` - 联系方式/在线咨询
- `/about` - 关于我们
- `/sitemap` - 网站地图

### 内容标准（GEO优化）
- 每个页面 `generateMetadata` 生成SEO meta
- 每个页面 JSON-LD 脚本
- H2提问式标题（如"餐饮地推服务适合哪些场景？"）
- 内容前50词包含核心信息
- 表格/列表展示复杂信息（便于AI提取）

### 技术标准
- Next.js 16+ App Router，SSR优先
- Tailwind CSS + Shadcn UI
- 移动端按钮点击区域 ≥48x48px
